# Teufel AI Control Center - Demo

This repository contains an interactive frontend demonstrator built for the **Technical AI Inhouse Consultant** role presentation at Teufel.

## 🚀 Live Demo
**[View the Live Application Here](https://Syeddan08.github.io/teufel-ai-demo)**

## 🎵 Features Included 

1. **IT-Business KPI Dashboard (Home)**: View dynamic operational metrics mapped to DSP Firmware and Acoustics R&D. Includes interactive Recharts comparing R&D Cost Savings vs AI Expenditures.
2. **Model Evaluation Center**: A side-by-side benchmark contrasting GPT-4o, Claude 3.5 Sonnet, and Llama 3 focusing on Audio Accuracies, equalizations, and latency thresholds.
3. **Secure RAG Pipeline Explorer**: An interactive chat simulation drawing responses specifically mapped to Teufel technical manuals (like ROCKSTER AIR 2 and ULTIMA 40 Surround). The interface renders the exact embedded knowledge chunks utilized.
4. **Observability Console**: Real-time simulated telemetry tracking API latencies natively tailored to audio tuning requests and error endpoints (like `BT-MIC-DROP.wav`).
5. **Governance Hub**: Outlines GDPR, NIS2, and EU AI Act constraints and active privacy setting checkpoints directly tailored to current hardware constraints.

## 🛠️ Technology Stack
- **Framework:** React + Vite
- **Styling:** Premium Native CSS (Teufel Red `#EB0000`, Inter Font, Glassmorphism variants)
- **Visuals:** [Recharts](https://recharts.org/) & [Lucide-React](https://lucide.dev/)
- **Deployment:** GitHub Pages 

## 💻 Local Execution
1. Clone the repository 
2. Install dependencies: `npm install`
3. Start the dev server: `npm run dev`
4. Access `http://localhost:5173`
